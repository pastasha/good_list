Для страницы "нужна помощь":
    массив. Содержимое каждого элемента массива:
        id - номер элемента
        message - текст типа помощи. Например: Помощь бездомным животным;Социальные столовые и раздача еды
        category -  url. При нажатии на категрию открывается страница с заданным url. То есть если category='food', то ссылка будет /need-help/food
    Пример вывода api:
    [{"id":0,"message":"Социальные службы, телефоны доверия, горячие линии","category":"tels"},{"id":1,"message":"Помощь бездомным животным","category":"animals"},{"id":2,"message":"Социальные столовые и раздача еды","category":"food"},{"id":3,"message":"Благотворительные фонды","category":"fonds"},{"id":4,"message":"Помощь временно перемещенным лицам","category":"moved"},{"id":5,"message":"Размещение информации на ресурсе","category":"post-info"}]
